## 實際行程
### Day 1

1000 武陵山莊出發

![池有名樹](/assets/images/20230910/IMG_5156.jpeg){: width="450" }
<br>
池有山開始的陡上, 同行夥伴回眸一笑.

1400 經過三叉營地, 開始下小雨. 從登山口走到這爬升已經破千了, 花了約3個小時, 沒特別催馬力, 算是輕鬆愉快的速度.

![池有名樹](/assets/images/20230910/IMG_5159.jpeg){: width="450" }
<br>
白牆版的池有名樹... :(
<br><br>
往新達山屋的路況比我想像中還難一些, 本來以為是平緩好走的稜線山徑, 結果路上都是大大小小的石瀑跟高低繞的山徑, 加上霧雨綿綿, 走的我都有點心煩...

1530 抵達新達山屋
![新達山屋](/assets/images/20230910/IMG_5161.jpeg){: width="600" }
<br>
接近日落時的新達營地, 可惜太陽被池有山擋住了, 不然這邊是一個不錯的營地.
<br>
<br>
後續於山屋中陸陸續續遇到一群單攻的大哥大姐們, 全身都溼透了, 幸好有一位女山友幫忙燒熱水跟零食幫忙補給, 要不然一旦失溫就會很危險. 
晚上約8點多時又有一組人從桃山過來, 也是全身都溼透了, 晚上摸黑到這甚是辛苦.
<br>
本來計畫在外面紮營, 但是遇到下雨, 就打消念頭住回山屋, 人多難免彼此干擾, 當晚果然沒睡好.

### Day 2

0700 起登

![新達山屋](/assets/images/20230910/IMG_5163.jpeg){: width="600" }
<br>
早上天氣大好, 陽光普照, 天空湛藍, 一掃昨天的陰雨綿綿.

![池有山](/assets/images/20230910/IMG_5165.jpeg){: width="600" }
<br>
好天氣的池有山跟新達池, 新達池是一個看天池, 還不小, 但是水質不是很好, 除非不得已才會去取用.

![大壩群峰](/assets/images/20230910/IMG_5166.jpeg){: width="600" }
<br>
從山屋後面就可以看見此行的目標, 右邊的山頭就是東霸第四峰, 東霸尖山, 從這邊開始路徑就一路抖下到塔克金溪底.

![塔克金溪](/assets/images/20230910/IMG_5170.jpeg){: width="450" }
<br>
秀霸線下塔克金溪後, 再陡上接回大壩線, 沿路上的景觀在陽光的照耀下, 整個森林都閃閃發光.

![塔克金溪-2](/assets/images/20230910/IMG_evan_yang.jpeg){: width="450" }
<br>
同行夥伴拍的, 在陽光照耀下, 綠到極致的青苔.

![霸南山屋](/assets/images/20230910/IMG_5173.jpeg){: width="450" }
<br>
大約11點就抵達霸南山屋, 早早煮完午餐就泡茶喝. 大鹿林道因崩塌封閉, 所以今天就由我們包下整個山屋.
### Day 3
一大早約4點左右出發, 對我而言有點早, 獨攀時其實都會等到天亮才出發. 途中經過的地方因為摸黑, 所以都沒什麽印象, 其實我不是很喜歡這種爬山方式, 不過一般登山客似乎都習以為常, 這也是我比較偏好獨攀的原因之一.

![ILoveYou平台](/assets/images/20230910/IMG_5179.jpeg){: width="600" }
<br>
抵達ILoveYou平台時才5點出頭.

![下切](/assets/images/20230910/IMG_5183.jpeg){: width="450" }
<br>
從IloveYou平台到東霸下切口還要走一小段路, 路跡不是很明顯, 花了一點時間才找到正確的入口. 下切口往下看, 那深深的谷底, 深吸一口氣就直接下去.

![down](/assets/images/20230910/down.jpg){: width="600" }
<br>
一開始是很細很細的碎石, 可以一路嚕下去, 我看網路上的影片想依樣畫葫蘆想很久了. 不過回程時這一段就慘了, 走兩步退一步, 到後來只能扶著旁邊的岩壁走.

![飛瀑](/assets/images/20230910/IMG_5188.jpeg){: width="450" }
<br>
到中段就會遇到從邊大壩流下來的飛瀑, 甚是壯觀, 可惜太陽沒照進來.

![金色山頭](/assets/images/20230910/IMG_5189.jpeg){: width="450" }
<br>
下切後段石頭也逐漸的變大, 也容易滾動, 跟隊友就切不同的路徑下, 避免踢下的滾石造成危險. 這時陽光也照上了旁邊的山頭, 黃金大壩.

![匯流處](/assets/images/20230910/IMG_5201.jpeg){: width="600" }
<br>
切到底抵達雙溪匯流處, 中間偏左邊的山頭就是目的地. 從這邊就要過溪, 看見東霸尖山的登山口開始陡升.

![石壁](/assets/images/20230910/IMG_5191.jpeg){: width="450" }
<br>
這一段路一開始混雜了台灣中級山林相雜亂的特色, 然後中間不算短的滾石碎石坡, 跟一些需要點抱石技巧的地形, 路徑也不是很清楚, 偶爾還要對照 gps 找路, 整體上不是很好走. 

![菇菇一家](/assets/images/20230910/IMG_5192.jpeg){: width="600" }
<br>
途中休息時, 遇到猴板凳一家七口.

![pre-peak](/assets/images/20230910/pre_peak.jpg){: width="450" }
<br>
上了東霸的稜線後, 箭竹密佈, 回程時還在這邊走叉了路, 小小迷路了一下.

![peak](/assets/images/20230910/peak.jpeg){: width="600" }
<br>
經過快3個小時的努力, 在早上9點多終於到達山頂, 累...

![peak-2](/assets/images/20230910/peak-2.jpg){: width="600" }
<br>
只能跪了... XD

![peak-2](/assets/images/20230910/IMG_5194.jpeg){: width="600" }
<br>
從這個角度看大小霸, 大概只有爬上東霸的人才能欣賞到吧 :)

怎麼來的就要怎麼回去, 剛剛所有經過的地形都要倒帶一次. 不同的是, 困難的依然困難, 容易的確變成難的. 雖然是下山, 但是花的時間沒有比上來少多少, 而且還要時時注意是否走過頭.

![back](/assets/images/20230910/IMG_5204.jpeg){: width="450" }
<br>
下切到匯流口後已經中午了, 就跟夥伴找地方休息吃乾糧, 準備最後的上切600m. 大約半小時後出發, 花了1個多小時慢慢的爬回到早上的飛瀑區. 從凌晨4點到現在, 大家也都累了, 所以我每一百公尺就休息一下, 希望能在下午後雷陣雨之前爬完這一段.

![back](/assets/images/20230910/IMG_5208.jpeg){: width="450" }
<br>
最後的微小碎石坡已經不能直接走上去了, 走三步退兩步, 只能扶著岩壁冒著落石的風險慢慢而上. 終於在2點半爬回當初下切的巨石, 終於完成這一趟困難的行程, 興奮的大喊.

![back](/assets/images/20230910/complete.jpg){: width="450" }
<br>
爬上來後決定跳上下切的這塊巨岩拍照留念.

最後再花了約1個半小時趕回霸南山屋, 抵達山屋大約3點多, 足足花了11個小時來回. 12多公里的行程, 爬升超過1200m, 加上一堆地形, 疲憊感很高, 在山屋坐著坐著就睡著了.

### Day 4

這一天就是單純的原路來回, 天氣大好, 不知不覺的加快腳步, 早上5點出發下午約2點就抵達車子旁. 然後就去露營區盥洗了一番後回程.

### 後記

1. 天氣, 跟之前預估的差不多, 第一天霧雨, 之後漸入佳境, 天氣大好.
2. 氣溫約15度上下, 晚上約10度, 穿著短袖羊毛底層衣加上 Hike Right 防曬衣行進剛剛好.
3. 吃完的包裝還蠻重的, 約100g重, 這部份可以將包裝事先拆掉來改善, 但是犧牲了保存的品質.
4. 吃的部份, 台酒泡麵大推. 堅果依然沒吃完, 大概是之前只吃堅果吃到怕了. 鱈魚香絲可. 早上咖啡加上 whisky 很棒, 驅寒又有酒香.
